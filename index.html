Frequently Asked Questions

At Diva Hosting, we understand that you may have questions about our services. To help you out, we have compiled a list of frequently asked questions that should provide you with the information you need. If you have any additional questions or concerns, please don't hesitate to reach out to our support team.

I can't install node modules because of the ram limit

Make sure you have package.json file with all of your packages you want to install then press "Reinstall" button available in the "Settings" tab on your server.

I can't install python modules because of the ram limit

Make sure you have requirements.txt file with all of your packages you want to install then press "Reinstall" button available in the "Settings" tab on your server.

How can I upload my files?

First compress your files into an archive (zip, tar, rar) after that upload the archive to the file manager on the panel. Then right click on the uploaded file and click "Unarchive". And you will have your files uploaded. You can also use sftp

Why is my account not creating?

Check your username. It should have 3+ characters in it, and cannot use any special characters.
Check your email. If it is being used, it won't create your account.
If none of these are the case, contact support.

My file is over 100MB!

Make sure you didn't include your node_modules folder or your .git folder, as these take up a lot of space. If your code is still above 100mb, you're doing something wrong. We would recommend you use sftp.

How can I change my password?

In the panel you can login with discord, or you can use /password-reset to get a new password for your account.

How do I install my packages manually?

For node.js, run npm i package-name. For Python, run pip install package-name. This will work if your startup command is set to bash.

How can I link my domain to my server?

To link your domain to your server, please open a ticket using /ticket, and we will assist you in proxy the domain for you.
